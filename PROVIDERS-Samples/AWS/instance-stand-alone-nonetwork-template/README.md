Commands Examples
=================

```
- terraform init
- terraform apply (Launch Environment)
```

Required Variables
=======================

```
- codcliente
- env (SML/PRD/TRN/CS1/CS2)
- iniciaiscliente
- envnumber
- instance_type_linux
- instance_type_windows
- key_name (rsa value from public key)
- tag_shutdown
- tag_startup
- AWS_ACCESS
- AWS_SECRET
- AWS_REGION
- dns_name (Domain Registered in Route53)
- Nome_VPC
```
